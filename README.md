# Movie Information Website

A simple web-based application developed for practice to display trending movies, provide trailer links, and allow users to search for movies. The project focuses on handling structured data, organizing digital content, and presenting information in a clean and user-friendly manner.

---

## Features

- Displays trending movies
- Search functionality to find movies by name
- Trailer links for selected movies
- Organized presentation of movie metadata
- Responsive and clean user interface

---

## Technologies Used

- React.js  
- Pure CSS  
- External Movie APIs for data retrieval

---

## Project Purpose

This project was created to practice:
- Handling and displaying structured and semi-structured data
- Managing digital resources such as media links and metadata
- Implementing search and filtering functionality
- Building reusable UI components using React

---

## How It Works

- Movie data is fetched from an external API
- Data is processed and displayed dynamically on the UI
- Users can search for movies using keywords
- Each movie entry includes relevant details and trailer links
